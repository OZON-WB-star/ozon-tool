# 权限设计

## 角色

### super_admin
系统最高管理员，也就是你自己。可以查看所有用户、所有店铺、修改套餐、停用用户、查看所有同步任务、管理系统配置。

### admin_staff
内部运营人员。可以查看用户、协助配置店铺、查看同步状态，但不能修改系统核心配置。

### user
普通用户。只能查看自己的店铺、商品、订单，只能使用已开通功能。

## 前后台隔离规则

- `user-frontend/` 是用户前台。
- `admin-frontend/` 是管理后台。
- 后端接口必须检查登录身份。
- 管理接口必须检查 `super_admin` 或 `admin_staff` 角色。
- 不能只靠前端隐藏菜单，后端也必须做权限校验。
