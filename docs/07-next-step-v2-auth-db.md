# V2 已完成：账号、权限、数据库

## 为什么先做这一步

正式 ERP 不能只靠静态页面。必须先解决：谁登录、谁能看什么、数据存在哪里、后台和前台如何隔离。

## 当前权限

- user：前台用户，只能看自己的店铺和商品
- admin：后台管理员，可以看所有用户、店铺和同步任务
- staff：预留内部员工角色

## 当前数据库

默认使用 SQLite：`backend/dae_erp.db`。
后续上线可切换 PostgreSQL，只要修改 `.env` 或环境变量 `DATABASE_URL`。
