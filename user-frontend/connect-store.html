Ozon API Key 获取位置

进入 Ozon 卖家后台，打开设置里的 Seller API / API Keys，生成 API Key 并复制 Client ID。
注意：API Key 只会在创建时显示一次。正式上线前建议使用 HTTPS、生产级密钥加密、操作日志和权限审计。
